KALİ LİNUX HAQQINDA

Kali, Linux Debian bazalı, CD/DVD-dən işə salına bilən xüsusiyyətli BackTrack qurucuları tərəfindən 2013cü ildə ərsəyə gətirilmiş olan bir təhlükəsizlik nəzarət sistemidir. Offensive Security Ltd. Vasitəsilə Mati Aharoni, Devon Kearns və Raphael Hertzog tərəfindən təkmilləşdirilməkdə və maliyyələşdirilməkdədir. İçərisindəki alətlər sayəsində bir çox sahədə (şəbəkə, əməliyyat sistemləri, Arduino və s.) təhlükəsizlik testləri etmək və program təkmilləşdirmək mümkündür. Desktop şəraiti olaraq BackTrackdəki kimi KDE seçimi yoxdur, 32 bit (i386), ARM və Armel struktur dəstəyi də var. Sadəcə GNOME desktop interfeysi istifadə seçimi vardır. Üstəlik 64 bit (amd64), 32 bit (i386), ARM və Armel struktur dəstəyi də var.Kali Linux, komputerlərin sərt diskinə yüklənə bilinər, bir CD vəya USB-dən live olaraq işə salına bilinər. Hətta virtual maşınlarda belə qurulub, istifadə oluna bilinər. Kali Linux, Debian əsaslıdır. Kalinin istifadə etdiyi paketlərin çoxu Debian program qaynağından əldə edilir.

Kali Linuxun sahib olduğu alətləri qısaca bu şəkildə sinifləndirə bilərik :

Yenidən məlumat əldə etmək, Qeyri-qanuni istifadə alətləri (exploitation tools), Axtarış alətləri (scanners), şifrə hücum alətləri, Aldatma (spoofing) və “qoxulama” (sniffing) alətləri, Naqilsiz hücum alətləri, Hüquqi alətlər, Cihaz qırıcılar, Backtrack xidmətləri, Tərsinə mühəndislik və s.
